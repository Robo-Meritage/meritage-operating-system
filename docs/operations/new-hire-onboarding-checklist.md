# New Hire Onboarding Checklist (System-Aligned)

**Classification:** Internal
**Category:** Operations / HR
**Status:** Active
**Last Updated:** 2026-07-02

## Purpose

A ready-to-use checklist for onboarding a new hire into how Meritage actually operates — not just tool access, but alignment to a system with defined phases, roles, and handoffs. For the underlying systems and rules, see the [Systems & File Management Standard](../technology/systems-and-file-management-standard.md).

---

## 1. Account Access (Day 1)

- [ ] Google Workspace (Drive, email, calendar)
- [ ] MadeMarket
- [ ] PandaDoc
- [ ] ROAM
- [ ] Suralink — case-by-case, not default (granted per deal when needed)
- [ ] Confirm access to: Active Deal folders, Financial Worksheets, Teaser Data Room templates (all in Drive)

## 2. Understand the System (Required)

- [ ] Google Drive = all work and files (source of truth)
- [ ] MadeMarket = deal tracking and buyer activity
- [ ] PandaDoc = NDAs and agreements
- [ ] Suralink = diligence, invite-only
- [ ] ROAM = internal communication
- [ ] Review the Meritage deal lifecycle (see [M&A modules](../m-and-a/README.md))

## 3. Understand Your Role

- [ ] Identify your lane: Deal Principal / Execution Team / Admin
- [ ] Confirm your responsibilities within each deal phase
- [ ] Identify who you report to and support

## 4. Core Rules (Non-Negotiable)

- [ ] Google Drive is the source of truth
- [ ] MadeMarket is for deal tracking only
- [ ] No documents shared without an executed NDA
- [ ] No files stored locally
- [ ] No duplicate files or versions
- [ ] Follow the [standard deal folder structure](../technology/systems-and-file-management-standard.md#standard-deal-folder-structure-google-drive) exactly; do not rename folders

## 5. Financial Workflow

- [ ] Locate Financial Worksheets in Drive; review the standard model
- [ ] Be able to calculate: Revenue, COGS, EBITDA, Adjusted EBITDA
- [ ] Save all work inside the deal folder only

## 6. Phase-Based Responsibilities

- [ ] **Origination / Onboarding** — confirm deal created in MadeMarket and Drive folder linked; attend onboarding call; understand deal strategy
- [ ] **Financial Analysis** — collect financials; build model in Drive; complete EBITDA / Adjusted EBITDA / valuation support
- [ ] **Go-To-Market** — assist CIM draft and financial section; prepare teaser materials; support buyer targeting
- [ ] **Buyer Engagement** — track NDA status; support buyer communication tracking and Q&A
- [ ] **Negotiation** — support LOI tracking; analyze offers; track deal structure changes
- [ ] **Due Diligence** — organize documents; track diligence requests; support data room (Suralink if activated)
- [ ] **Closing** — support closing checklist; confirm final documents stored in Drive; support wire and final coordination

## 7. First 14-Day Ramp Expectations

**By end of Week 1:** navigate Drive and deal folders independently; understand the full deal lifecycle; build a basic financial worksheet; understand the NDA + CIM process.

**By end of Week 2:** contribute to an active deal; support CIM and teaser prep; track buyers in MadeMarket; participate in execution calls.

## 8. Acknowledgment

- [ ] I understand the Meritage deal system and my role within each phase
- [ ] I will follow the Drive folder structure and document-control rules
- [ ] I will not share materials without an executed NDA
- [ ] I understand Google Drive is the source of truth
